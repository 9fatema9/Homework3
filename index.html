Команды для Git

git --version
Проверка текущей версии програмы.
git confing --global user.name "Your Name"
Если вы никогда не использовали git, то данный параметр используется для установки вашего имени.
git config --global user.email "name@domain.example"
Данный параметр используется для установки вашей электронной почты.
git init
Команда для создания репозитория, которая выполняется 1 раз для всего проекта.
git add
Индексирование для созданныхфайлов и их изменений.
git commit -m "комментарий"
Фиксирование всех сохранённых изменений.
git status
Проверка текущего состояния репозитория.
git log
Проверка текущей версии програмы.
git checkout
Переключение между коммитами.
git remote add origin https/...
Создание связи между GitHub репозиторием и репозиторием на компьютере.
git push origin master
Данный будут загружены на репозиторий в GitHub
